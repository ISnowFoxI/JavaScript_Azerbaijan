Giriş

JavaScript müxtəlif platformalarda istifadə olunan, öbyekyönümlü skriptləşdirmə dilidir. Kiçik və yaddaşda az həcmli dildir. JavaScript Host mühitində JavaScript, həmin mühitdə olan obyektlərlə əlaqələndirilir ki, bunula da həmin obyektləri proqram kodları vasitəsi ilə idarə edir.

JavaScript özündə obyetklərin standart kitabxanasını, yəni Massiv(Array), Zaman(Date) və Riyazi(Math) kimi kitabxanaları birləşdirir. Bundan əlavə operatorlar, idarəetmə strukturları və əmrlər kimi əsas dil elementlərini birləşdirir. JavaScriptə əlavə obyektlər daxil etməklə onu müxtəlif məqsədlər üçün istifadə etmək olar. Məsələn

  • Müştəri tərəfi JavaScript, Document Object Model (DOM) və brauzeri idarəetmək üçün obyektlər təmin edərək dili daha da genişləndirir. Müştəri tərəfi əlavələr proqrama HTML form-unda elementlər əlavə etməyə, istifadəçi tərəfindən işə salınan hadisələrə(events) (mouseun düyməsini basmaq), forma məlumat daxil etmək, səhifədə gəzinmək və.s) cavab verməyə imkan verir.
  • Server tərəfi JavaScriptin serverdə istifadə ediləbilməsini təmin edən obyektlərlə təmin edərək dili genişləndir. Server tərəfi genişlənmələr applikasiyanın verilənlər bazası ilə əlqəsini, serverdə yerləşən fayllara dəyişiklərin edilməsini, müştəri-server arasında informasiyanın ötürülməsində ardıcıllığın təmin edilməsi və.s təmin edir.
Bilməli olduqların

Bu bələdçidə aşağıdakılar haqqında məlumatın olduğu qəbul olunmuşdur:

  • İnternetin və World Wide Web(WWW) haqqında ümumi fikirlər.
  • HTML ilə işləmə bacarığı.
  • Proqramlaşdırma bacarığı. Əgər proqramlaşdırmaya yeni başlayırsansa onda freeCodeCamp.org da olan JavaScript dərslərinə nəzər sal.
JavaScript və Java

JavaScript və Java bəzi hissələrdə bir-birinə bənzəyir, digər hissələrdə isə fundemental olaraq fərqlənirlər. JavaScript dili Javanya bənzəsə də, Javanda olan static və strong typing checking JavaScriptdə yoxdur. Lakin JavaScriptdə olan ifadələr sintaksisi, adlandırma qaydaları və sadə idarəetmə axını konstruksiyası Java ilə eynidir. Bu səbəbdən də, adı LiveScript-dən JavaScript-ə dəyişdirilmişdir.

JavaScript və Java

JavaScriptlə kod yazmaq üçün, js.do açın və aşağıdakı "Salam Dünya" kodunu yazın.

function greetMe(variable) { alert("Salam " + variable); } greetMe("Dünya");

Kodu yazdıqdan sonra "Run" düyməsinə basın və nəticə ekranda çıxacaq.
